# 🍔 JungleBite - 餐飲外送平台系統

## 📌 專案介紹

JungleBite 是一個模擬外送平台的網站系統，整合會員、商家與外送員等多種角色，提供餐廳與商品瀏覽、電子錢包、交易管理與優惠券機制等功能。  

本專案從資料庫設計到前後端整合，完整實作一個具備基本電商與平台營運邏輯的系統，並進一步結合資料蒐集與人臉辨識功能，提升系統的完整性與互動性。

---

## 🚀 核心功能

### 👤 使用者與角色系統
- 多角色設計：Customer / Merchant / Delivery Person  
- 使用者登入與身分識別  
- 個人資料與帳戶管理  

### 🛍️ 餐飲平台功能
- 餐廳與商品資訊瀏覽  
- 商家資料整合與展示  
- 基本訂單與交易流程（模擬）  

### 💰 電子錢包系統
- 錢包餘額管理  
- 儲值 / 提款功能  
- 完整交易紀錄查詢  

### 💳 卡片與支付管理
- 銀行卡片綁定  
- 多帳戶管理  
- 支付流程整合  

### 🎟️ 優惠券與活動系統
- 折扣優惠券發放  
- 優惠券使用與過期機制  
- 使用紀錄管理  

### 🎮 遊戲化機制
- Web 小遊戲（Flappy Bird）  
- 根據遊戲分數產生折扣  
- 每日遊戲次數限制  

---

## 🤖 進階功能

### 🧠 人臉辨識登入
- 支援人臉辨識作為登入方式  
- 提升使用者驗證便利性  

### 🕷️ 商家資料爬蟲
- 自動蒐集外送平台餐廳資訊  
- 整理營業時間與資料結構化

## 🧱 系統架構概念

系統主要模組包含：

- User（Customer / Merchant / Delivery）  
- Wallet（電子錢包）  
- Card（銀行卡）  
- Transaction（交易紀錄）  
- Coupon（優惠券）  

透過模組化設計，讓各功能可以獨立運作並相互整合。
